Enumerable methods in Ruby: map, group_by, and cycle

October 30th, 2014

This week's challenges have definitely been a lot more time-consuming; I don't think I'm having any logical difficulties, but the hardest part has been researching Ruby methods and picking the most appropriate one to use. Here are some of Ruby's enumerable methods and how to use them:

Enumerable#map

The map method is called on something to be enumerated (an array, hash, range, etc.), runs the following code block once for each element, and returns the results in an array. For example,

(1..4).map { |i| i**2 }

will return

[1, 4, 9, 16]

In this example, each integer in the range 1..4 is squared.

Enumerable#group_by

The group_by method returns a hash that groups elements according to their evaluated values.

(1..6).group_by { |i| i % 3}

returns

{ 0 => [3, 6], 1 => [1, 4], 2 => [2, 5]}

In this example, a hash is returned that groups elements by their modulus, where the modulus is the key and the element is the value.

Enumerable#cycle

cycle takes an integer n and will repeat the code block n times. If n is not specified, then the code block will repeat forever.
