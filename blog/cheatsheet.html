What is SQL and what is it used for?

SQL pronounced "sequel" stands for Structured Query Language. It is a standard interactive and programming language for getting information from and updating a database. SQL lets you access and manipulate databases.

Useful SQL commands:

SQL SELECT:Selects data from one or more database tables.
Ex1:
SELECT * FROM Table (will display all fields)
Ex2:
SELECT ColumnName1, ColumnName2
FROM Table1 (will display data from ColumnName1 and ColumnName2)

SQL INSERT INTO: insert new records into a SQL table.
Ex1:
INSERT INTO table_name (column1,column2,column3,...)
VALUES (value1,value2,value3,...); (will insert values under corresponding columns)

SQL WHERE: displays data using a conditional statement. Is used with other commands such as SELECT, FROM, etc.
The conditional statements that can be used are: <, >, =,etc.
Ex1:
SELECT column_name,column_name
FROM table_name
WHERE column_name operator value;

SQL UPDATE:updates data in the database table.
Ex1:
UPDATE table_name
SET column=new_value
WHERE column=old_value; (will replace old_value with the new_value)

SQL DELETE: will delete data from the database table.
Ex1:
DELETE FROM table_name
WHERE column=value; (will delete record at column=value; if WHERE isn't specified, all records will be deleted)

SQL ORDER BY:in what order to display a data set retrieved by a SQL SELECT statement.
Ex1:
SELECT column_name
FROM table_name
ORDER BY column_name ASC|DESC; (ASC= ascending order;DESC=descending order)

SQL LIKE: used to search for a specific pattern in a column.
Ex1:
SELECT column FROM table
WHERE record LIKE pattern;
Different patterns:
'%s' will find records that end with 's'
's%' will find records that begin with 's'
'%s%' will find records that include 's'
