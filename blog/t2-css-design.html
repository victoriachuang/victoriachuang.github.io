Week 2: CSS

October 17th, 2014

This week's content was focused on HTML and CSS. Here are some design-related topics that I did some research on:

Margin vs. border vs. padding

  • I found this great image on W3Schools that explains the differences between margins, borders, and padding. Imagine your content to be contained in a box. The padding creates some space between the content and the border. Stylistically, increasing padding will make your content look less cramped. The border is the box itself; it can be styled different ways, with options for width, color, and style (ie. dotted, dashed, etc.). The margin is the space between the box and other objects. Increasing margins, just like padding, can make your document look more balanced, especially if you have multiple objects on the page.

Best practices for using tables vs. divs

  • Tables and divs are both used to display information in an organized manner on a document, but it can be hard to decide which one to use. On Stack Overflow, I found a great answer as to when to use each one: if you would write it in Microsoft Word, use a div; if you would display it in Excel, use a table. A table is also best used to contain tabular data, such as data that would be best represented in a spreadsheet or grid.
